## üîß Troubleshooting PDF Generation

### Common Issues & Solutions

**Issue 1: "pandoc: command not found"**
```bash
# Solution: Install pandoc (see installation guide above)
# Verify installation:
pandoc --version
```

**Issue 2: "xelatex not found"**
```bash
# Solution: Install LaTeX distribution
# macOS: brew install --cask mactex
# Windows: Download MiKTeX from https://miktex.org/
# Linux: sudo apt-get install texlive-xetex texlive-fonts-recommended
```

**Issue 3: "PDF generation failed"**
```bash
# Solution: Try with different PDF engine
pandoc *.md -o output.pdf --pdf-engine=pdflatex
# Or try without complex options:
pandoc *.md -o output.pdf
```

**Issue 4: Large file size**
```bash
# Solution: Compress PDF after generation
# macOS: Use built-in compression
# Linux: gs -sDEVICE=pdfwrite -dCompatibilityLevel=1.4 -dPDFSETTINGS=/screen -dNOPAUSE -dQUIET -dBATCH -sOutputFile=compressed.pdf input.pdf
```

**Issue 5: Formatting issues**
```bash
# Solution: Try HTML conversion first
pandoc *.md -o architecture.html
# Open in browser and save as PDF
```
